Class MapServer

MapServer类，设置地图不同版本的类，此对象配合EzMap.addVersion方法和EzMap对象的实例方法showVersion使用

构造说明：参数为地图图片引用地址，其类型可以是String类型或String的Array类型，如:[ "http://172.26.15.21:8088/EzServer","http://172.26.15.21:8088/EzServer"]。

例如：
uMapServer = MapServer("http://192.168.10.94:8888/EzServer","http://192.168.10.15:8888/EzServer")
EzMap.addVersion("2005",uMapServer);
uEzMap.showVersion("2005");// uEzMap为一个EzMap实例

说明：
在EzMapAPI.js中配置的EzServerClient.GlobeParams.MapSrcURL在初始化时也是构建一个默认MapServer对象，其默认的版本信息为"defaultMapVersion"
例如下边方法可以显示初始版本地图：
uEzMap.showVersion("defaultMapVersion")//uEzMap为EzMap对象实例

Defined in: MapServer.js.
