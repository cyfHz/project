Class EzGeoPSTool

EzGeoPSTool类，是一个静态类，提供一些静态方法来调用EzGeographicProcessingService服务来进行空间几何关系判断和几何运算（需要在EzMapAPI.js中设置地理处理服务的url地址才能调用这些方法）
Defined in: EzGeoPSTool.js.
