Class EzTileThematicCP03

EzThematic类，表示一个调用cp03的动态瓦片专题图类 构造说明：构造函数中的参数指向专题图服务请求专题图的URL，注意几个参数的设定
举例：
var uEzThematic = new EzTileThematicCP03("http://192.168.10.192:8888/BlankArcServer/servlet/agsStatistic", "GIS0001.KC_JWXQ_PCS_PGDMPieChart50T_SGFX_GFLB20100506101505687zzjgdm入室盗窃02550119.4184570312531.50830078125120.3803710937532.0913085937510]]>df123800800truepng500500", false, "http://192.168.10.183:8888/EzServer/Proxy");
//在地图中展现专题图
uEzThematic.open(uEzMap);//uEzMap为EzMap的一个实例
