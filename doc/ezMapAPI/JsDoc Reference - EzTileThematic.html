Class EzTileThematic

EzThematic类，表示一个动态瓦片专题图类 构造说明：构造函数中的参数指向专题图服务请求专题图的URL，注意几个参数的设定
举例：
//注意EZBOX、EZWIDTH和EZHEIGHT三个占位符
var uEzThematic = new EzTileThematic("http://192.168.10.185:8399/arcgis/services/pg/MapServer/WMSServer?service=WMS&VERSION=1.3.0&REQUEST=GetMap&LAYERS=0&CRS=CRS:84&BBOX=EZBOX&WIDTH=EZWIDTH&HEIGHT=256&FORMAT=image/png&TRANSPARENT=TRUE", false, "http://192.168.10.183:8888/EzServer/Proxy");
//在地图中展现专题图
uEzThematic.open(uEzMap);//uEzMap为EzMap的一个实例

EzTileThematic(vFormatURL, vBIsStreamOrText, vThematicProxyURL) 
new EzTileThematic(String vFormatURL, Boolean vBIsStreamOrText, String vThematicProxyURL);
Parameters:
{String} vFormatURL
参数含义：带有占位符的专题图请求URL，其中包含EZBOX、EZWIDTH和EZHEIGHT三个占位符，用于传递转换当前请求视角的地图范围
参数省略：不可以省略
{Boolean} vBIsStreamOrText
参数含义：当前MIME为文本内容时为false，以流方式获取专题图此值为true，省略情况下为文本方式
参数默认值：false
参数省略：不可以省略
{String} vThematicProxyURL
参数含义：以文本方式请求时，ajax请求代理地址,如果不存在ajax跨域时，此参数可以省略
参数省略：可以省略
